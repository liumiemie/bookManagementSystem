# bookManagementSystem

基于nodejs的express框架实现的简易图书管理系统

# 线上地址

http://120.78.142.165:2907/ 

# 安装

1.克隆项目地址并下载（需要安装Git）

git clone git@github.com:liumiemie/bookManagementSystem.git

2.通过npm安装本地服务第三方依赖模块（需要安装Node.js）

npm install

# 运行程序
执行node指令运行backEnd.js


node backEnd
